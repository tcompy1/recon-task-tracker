# Recon Task Tracker

A small full-stack tracking app built with React, Vite, TypeScript, Express, Prisma, and PostgreSQL.

## Project structure

- `client/` - frontend
- `server/` - backend
- `docs/` - planning and specs

## Root scripts

- `npm run dev:client`
- `npm run dev:server`
- `npm run lint`
- `npm run test`
- `npm run db:migrate`
- `npm run db:push`
- `npm run db:generate`
